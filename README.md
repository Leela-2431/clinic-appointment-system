# Clinic Appointment System

A web-based Clinic Appointment Management System developed to streamline patient registration, appointment booking, and profile management. This project focuses on backend logic, database integration, and user-friendly workflows.
---
## Technologies Used
- Node.js
- Express.js
- MySQL
- EJS (Embedded JavaScript templates)
- HTML, CSS, JavaScript
---
## Features
- Patient registration and login
- Book clinic appointments
- View upcoming and past appointments
- Update patient profile details
- Secure authentication
- Role-based views (Patient / Admin)
- Dynamic UI using EJS templates
---
## Project Structure
- `views/` – EJS pages (login, register, appointments, profile)
- `routes/` – Application routing logic
- `public/` – Static files (CSS, JS, images)
- `app.js / server.js` – Main server file
- Database SQL file included
---
## How to Run the Project (Localhost)
1. Install Node.js and MySQL
2. Import the provided SQL file into MySQL
3. Configure database credentials in the project
4. Open terminal in project folder
5. Run:
   ```bash
   npm install
   npm start
